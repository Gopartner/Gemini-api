# Gemini API implementasi web-app dan cli-app

## jalankan cli app:
```bash
npm run ai
```
## jalankan web app:
```bash
nom run dev
```

## clone repository:
```bash
https://github.com/Gopartner/Gemini-api.git
```

- setelah clone, cd Gemini-api
- jalankan perintah ini untuk instalasi
```bash
npm install
```
